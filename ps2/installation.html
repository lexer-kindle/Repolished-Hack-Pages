Installation Process

Read this first. Modding carries risk. Back up your memory card before you begin, and never interrupt power during a write to the card or console. MechaPwn in particular can permanently break the DVD player on pre-Deckard units, so treat it as optional and fully understand the consequences before running it. Only use this on hardware you own, with software and game backups you are legally entitled to. Proceed at your own risk.

The process below is a high-level roadmap. It contains no exploit code or payloads; you supply the official homebrew files yourself. Work through the phases in order and verify each one before moving on.

  1. Identify your boot ROM. Get temporary homebrew running and open uLaunchELF to read the boot ROM version on screen. If it reports 2.30 or 2.50, FMCB is unavailable and you will use the FreeDVDBoot route. Otherwise either route works.
  2. Bootstrap into homebrew. On the FMCB route, insert a prepared FreeMcBoot memory card so the launcher auto-boots at power-on. On the FreeDVDBoot route, burn the FreeDVDBoot image to a quality DVD-R at a slow speed, insert it, and let the DVD player launch uLaunchELF automatically.
  3. Install a persistent exploit. Use your bootstrapped session and the file manager (wLaunchELF) to write a permanent memory-card exploit. On compatible ROMs this is FMCB itself; on others, install PS2BBL or OpenTuna so homebrew survives reboots without the disc.
  4. Set up Open PS2 Loader. Copy OPL and your launcher configuration to the memory card. Prepare your chosen storage backend: format a USB drive as FAT32, set up an MX4SIO SD card, partition an internal HDD, or configure an SMB share. Place your game backups in the folders OPL expects.
  5. Optional region-free with MechaPwn. Only if you have weighed the DVD-player risk, run MechaPwn to rewrite the region flag. Skip this step entirely if you are unsure or value the built-in DVD player.
  6. Test everything. Reboot without the bootstrap disc to confirm the persistent exploit auto-loads. Launch OPL, load a game from your storage device, and check that saves write back to the memory card. If anything misbehaves, consult the troubleshooting page before changing multiple settings at once.

Once a clean reboot lands you in your launcher and a game loads from storage, your softmod is complete and ready for daily use.
