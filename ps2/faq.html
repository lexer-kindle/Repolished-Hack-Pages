PlayStation 2

FAQ

Frequently Asked Questions

Does modding my PS2 require opening it or soldering?

No. Everything in this guide is a softmod. It lives on a memory card or a burned disc, so there is no hardware modification, no soldering, and nothing permanent done to the console itself.

How do I know whether to use FMCB or FreeDVDBoot?

Check your boot ROM version in uLaunchELF. If it reads 2.30 or 2.50, FMCB will not install and you must use FreeDVDBoot, then add a persistent memory-card exploit. Any other supported ROM can use FMCB.

Can I undo the mod?

Yes. A memory-card exploit can be removed by deleting its save data or formatting the card, and a FreeDVDBoot disc simply stops being used. MechaPwn changes are the main exception, since they alter low-level console settings.

What is the best way to store and load games?

Open PS2 Loader reads from USB, an MX4SIO SD adapter, an internal HDD, or an SMB share. An internal HDD is fastest, USB is the most universal, and MX4SIO is a popular compromise.

Will a clone memory card work?

It might, but genuine Sony 8MB cards are far more reliable for holding an exploit. Many boot failures trace back to clone cards, so use an authentic one if you can.

Does MechaPwn break anything?

It can. On pre-Deckard units MechaPwn may disable the built-in DVD player. It is optional, used only for region-free play, so skip it if you value DVD playback.

Is region-free necessary?

No. Many users never need it. It only matters if you want to run discs from other regions, and OPL-loaded backups generally do not require it.
