Xbox One

Methods

The Method: Official Developer Mode

There is exactly one legitimate way to run your own software on an Xbox One, and it is Microsoft's own Developer Mode. Because no jailbreak exists, this page is short on alternatives and long on understanding the single path that actually works. The good news is that it is well documented, supported, and reversible.

How Developer Mode Works

A retail Xbox runs in a locked-down mode that only accepts signed, Store-distributed software. Developer Mode adds a second environment alongside that retail mode. When you switch to it, the console boots into a sandbox that is willing to run apps you provide yourself, as long as they are packaged in the format Microsoft expects. Switching modes does not wipe your games or saves; the two environments simply coexist.

The Account Layer

Access is gated by a Partner Center developer account. You register it through a web browser against your existing Microsoft account. As of June 2025, individual onboarding is free, so the only cost is a few minutes of form-filling. Once registered, your Microsoft account is authorized to flip a console into Developer Mode using the Dev Mode Activation app, a free download from the Store on the console.

UWP: The Software Format

Everything you sideload is a Universal Windows Platform (UWP) app. This is the same app model used across Windows and Xbox. Because the community can compile open-source projects into UWP packages, a healthy library of useful software already exists. The headline category is emulation:

  • RetroArch bundles dozens of emulator cores into one app, covering everything from the NES to the PlayStation Portable era.
  • Dolphin brings GameCube and Wii emulation, which benefits from the extra power of the One X and Series consoles.
  • PPSSPP handles PlayStation Portable titles with strong compatibility.

Sideloading via Device Portal

Once a console is in Developer Mode, it exposes a web-based Device Portal. From your computer's browser you connect to the console's address, sign in, and upload UWP packages directly. The portal also lets you launch, manage, and remove apps. No USB transfer dance is required; the whole pipeline runs over your local network.

Knowing the Limits

Developer Mode is generous but not unlimited. Each app is capped at 2 GB, the filesystem is sandboxed, and the platform deliberately cannot run pirated retail games. Within those bounds, however, it is a genuinely capable tinkering and emulation environment.
