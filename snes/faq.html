Super Nintendo

FAQ

Frequently Asked Questions

Do I need to update the SNES firmware before modding?

No. The Super Nintendo has no firmware or internal software to update. Every upgrade is hardware: a flashcart, a region mod, or a video mod. There is nothing to flash on the console itself.

What is the difference between the FXPak Pro and the SD2SNES?

They are the same product line. FXPak Pro is the current production name for what was previously sold as the SD2SNES. Both are FPGA flashcarts with near-full library support, special-chip compatibility, and MSU-1 audio.

Do I still need a region mod if I have an FXPak Pro?

Not for flashcart games. The FXPak Pro auto-patches region and ships in a multi-region shell. You only need a SuperCIC or Super CIC++ mod if you want to run original imported cartridges on the bare console.

How do I tell if my console is a 1CHIP?

Open the shell and read the motherboard silkscreen. A board marked SNS-CPU-1CHIP-01 or SNS-CPU-1CHIP-02 is a 1CHIP unit with cleaner native RGB. Other markings indicate an older multi-chip board.

Does my 1CHIP console need an RGB amp?

Not really. 1CHIP consoles already output clean RGB, so an amp gives only a marginal improvement. Older multi-chip boards benefit clearly, and the SNES Jr (SNS-101) requires an amp for proper RGB.

Can I run homebrew and MSU-1 hacks?

Yes. Both run as ROMs loaded from the FXPak Pro's SD card. MSU-1 packs add streamed, CD-quality audio to supported games, and the flashcart plays them directly with no console modification.

Is any of this reversible?

The flashcart is fully removable. Internal mods involve soldering and are reversible only with care, so weigh that before opening the shell.
