Compatibility

Model and Firmware Compatibility

Every retail PSP model can run modern custom firmware, which is part of why the platform remains so popular. The key requirement is getting the system onto the final official firmware, 6.61, before installing CFW. If your PSP shows an older version in the system settings, update it first using Sony's official network update or an official update file copied to the Memory Stick.

The hardware family breaks down as follows:

  • PSP-1000 ("Phat") — the original 2004 model. Fully supported, and the only model that can use the classic Pandora's Battery hardware unbrick method.
  • PSP-2000 ("Slim") — lighter, with more RAM. Some early units support hardware Pandora methods.
  • PSP-3000 — improved screen; software CFW only.
  • PSP Go (N1000) — no UMD drive, uses internal flash storage plus optional M2 card.
  • PSP Street (E1000) — the budget model with no Wi-Fi; still fully CFW-capable.

For the vast majority of users in 2026, the answer is the same regardless of model: update to 6.61, install ARK-4, then apply 6.61 Infinity to make it stick. Pandora's Battery is only relevant if your console is bricked or stuck on firmware too old to update normally. Always match your CFW files to the 6.61 firmware version — installers built for other versions will refuse to run or could cause errors.
