What You'll Need

Gathering everything before you start makes the install quick and stress-free. None of this is exotic, and most PSP owners already have the basics on hand. Here is the full checklist:

  • A PSP on firmware 6.61 — any model (1000, 2000, 3000, Go, or Street E1000). If yours is on an older version, you will update it to 6.61 as the first step.
  • A Memory Stick Pro Duo — this holds the CFW files, homebrew, and any game backups. A microSD card in a Pro Duo adapter works on most models. The PSP Go uses its internal storage but can also take an M2 card.
  • The official charger — keep the battery above roughly half charge throughout. A sudden power loss mid-install is the main avoidable risk, so plug in.
  • A computer — Windows, macOS, or Linux is fine. You only need it to download files and copy them to the Memory Stick.
  • A way to connect the Memory Stick to the PC — a USB card reader, or the PSP's own USB cable using its USB connection mode.
  • The CFW files — the latest ARK-4 (or 6.61 PRO) package and the 6.61 Infinity installer. Always download the current releases that match 6.61.

It is also worth setting aside about twenty to thirty minutes of uninterrupted time. The process itself is short, but rushing increases the chance of copying a file to the wrong folder. If your Memory Stick holds anything important, back it up to the PC first — formatting is sometimes recommended for a clean start. With these items ready, you are set to move on to choosing a method.
