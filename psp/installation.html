Installation

Read this first. Custom firmware is one of the safer console mods, but you still modify system software at your own risk. Keep the PSP plugged into its charger or above half battery throughout, never power off during an install step, and only use files that match firmware 6.61. Following the steps in order keeps the brick risk very low, but no guide can promise zero risk.

The full process moves from confirming your firmware to having permanent CFW with homebrew ready to go. Work through the phases below without skipping ahead.

  1. Check and update to 6.61. Open the system settings and look at the firmware version. If it is not 6.61, update using Sony's official network update or an official 6.61 update file copied to the Memory Stick. CFW installs only onto 6.61, so do not continue until you see that number.
  2. Prepare the Memory Stick. Connect it to your PC via a card reader or the PSP's USB mode. Back up anything important, then download the latest ARK-4 package and the 6.61 Infinity installer. A clean or freshly formatted stick avoids leftover-file confusion.
  3. Copy the CFW files. Extract the ARK-4 download and place its folders onto the Memory Stick exactly as instructed by the package, typically into the PSP/GAME directory. Putting files in the wrong folder is the most common mistake, so double-check before disconnecting.
  4. Run the installer. Safely eject the stick, return it to the PSP, and open the ARK-4 installer from the GAME menu. Let it run fully to load custom firmware. At this point CFW is active but still temporary — a full shutdown would undo it.
  5. Make it permanent with Infinity. While CFW is running, launch the 6.61 Infinity installer and complete it. This hooks the boot process so your CFW loads on every startup, including after a battery pull. Reboot to confirm CFW comes up automatically.
  6. Set up homebrew. With permanent CFW in place, copy your homebrew apps, emulators, and any plugins onto the Memory Stick. Launch one or two to confirm everything works, and your PSP is ready.

If a step does not behave as described — the installer refusing to start, or CFW vanishing after a reboot — stop and check the troubleshooting page before retrying. Most failures trace back to a version mismatch or a misplaced file rather than anything serious.
