PlayStation Portable Hacks Guide

Install Custom Firmware on virtually any PSP to run homebrew, emulators, and game backups from a Memory Stick.

What this is for

The PSP has a mature, friendly custom-firmware scene: CFW unlocks homebrew, emulators, ISO loading, and plugins. Nearly every PSP can run CFW, either permanently or per-boot, and the process is one of the safer console mods.

⚙️ Methods, versions & compatibility — read this first

MethodWorks onNotes
ARK-4 / 6.61 PRO (CFW)

CFW		Most PSP models on 6.61Makes CFW permanent so it survives reboots without re-running an installer.
Pandora's Battery / hardware methods
Hardware		Original PSP-1000 (and some 2000)Recovery/older method using a modified battery; mostly for bricked units or very old firmware.

What to check first: 6.61 is the final official firmware and the target for modern CFW. Most PSPs can update to 6.61 and run CFW; the PSP Street (E1000) and Go have specific supported CFW (ARK-4). Check your firmware under Settings - System Settings - System Information before starting.

⚠️ Before you start: Console modding carries real risk — an interrupted install or the wrong method/file for your exact model can leave hardware unbootable. Identify your model first, back up anything important, and proceed at your own risk.

What you'll need

  • A PSP and its charger
  • A Memory Stick Pro Duo (or microSD + adapter / internal storage on the Go)
  • A computer with a Memory Stick reader
  • The CFW files (e.g. ARK-4 / 6.61 PRO) for your firmware

How it works — the process

This walkthrough names the tools at each stage so you know what each one is and what it's for. Always use the latest release of each named tool.

  1. Check and update firmware
    Confirm your firmware in System Information; update to 6.61 (official) if needed, since modern CFW targets it.
  2. Copy the CFW files
    Place the CFW installer (ARK-4 / 6.61 PRO) onto your Memory Stick from a computer.
  3. Run the CFW installer
    Launch the installer from the PSP's memory to boot into custom firmware.
  4. Make it permanent (optional)
    Install 6.61 Infinity so CFW survives reboots without re-running the installer.
  5. Set up homebrew
    Add homebrew apps, emulators, plugins, and your game backups to the Memory Stick.

Which option should I pick?

For almost everyone: update to 6.61 and install ARK-4 (with Infinity for permanence). Reserve Pandora's Battery methods for very old firmware or unbricking.
