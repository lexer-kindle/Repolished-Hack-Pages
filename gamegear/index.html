Sega Game Gear Hacks Guide

Recap first to fix the sound and power, then add a bright IPS/McWill screen and a flashcart.

What this is for

Almost every unserviced Game Gear suffers from leaking surface-mount capacitors that cause distorted, low, or no audio and power faults. A full recap is the single most important fix and should come before anything else. Once stable, the dim original LCD can be replaced and games loaded from a flashcart.

What to check first: The Game Gear has no firmware or software exploit. The essential work is a full hardware recap to fix sound and power, after which optional quality-of-life mods (McWill or IPS LCD, EverDrive-GG flashcart) can be added.

⚠️ Before you start: Console modding carries real risk — an interrupted install or the wrong method/file for your exact model can leave hardware unbootable. Identify your model first, back up anything important, and proceed at your own risk.

What you'll need

  • A Game Gear (note model differences across revisions)
  • A capacitor kit (Console5, ZedLabz, or Retrosix SMD)
  • A soldering iron, flux, solder, and isopropyl alcohol
  • A McWill or IPS LCD replacement kit (optional)
  • An EverDrive-GG flashcart and SD card (optional)
  • Screwdrivers and a spudger to open the case

How it works — the process

This walkthrough names the tools at each stage so you know what each one is and what it's for. Always use the latest release of each named tool.

  1. Open and inspect
    Disassemble the Game Gear and check the SMD capacitors and sound-board connector for leakage and cold joints.
  2. Full recap
    Replace all electrolytic capacitors with a kit; this restores audio and power and is required before any other upgrade.
  3. Test the system
    Power on and confirm clean audio, stable power, and a working display before adding mods.
  4. Install a new screen
    Fit a McWill or IPS LCD kit; the LCD should be installed before other upgrades.
  5. Add a flashcart
    Use an EverDrive-GG to load Game Gear, Master System, and SG-1000 ROMs from SD.
  6. Optional extras
    Add controller-port or video-output mods for external display and capture.

Which option should I pick?

Recap before anything else - it's the #1 fix and a prerequisite for every other mod. Then choose McWill (proven, premium) or an IPS kit (cheaper) for the screen, and add an EverDrive-GG for flashcart loading.
