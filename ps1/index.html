PlayStation 1 Hacks Guide

Defeat the disc copy-protection so the console boots backups, imports, and homebrew from CD-R or an SD-card optical drive emulator.

What this is for

PS1 modding bypasses the console's licensed-media check so it will boot burned game backups, out-of-region imports, and homebrew. It lets owners preserve scratched or aging discs by playing copies, and run loose homebrew. Modern optical drive emulators go further by replacing the laser entirely and loading game images from an SD card.

What to check first: There is no software-only loader for PS1 - running backups requires hardware (a modchip or an ODE), except the swap trick on a few earliest fat models. PAL consoles have extra BIOS-level region protection, so modchips must connect to BIOS pins for out-of-region games. Confirm your exact board revision (PU-7/PU-8/PU-18/PU-22/PU-23/PM-41 etc.) before buying.

⚠️ Before you start: Console modding carries real risk — an interrupted install or the wrong method/file for your exact model can leave hardware unbootable. Identify your model first, back up anything important, and proceed at your own risk.

What you'll need

  • Your exact PS1/PSone board revision identified (open the console)
  • A soldering iron, fine wire, flux and a multimeter (for modchip installs)
  • A modchip (PSNee/MM3/OneChip) OR an ODE (xStation/MODE)
  • Quality CD-R media and a burner (modchip route)
  • An SD/microSD card (ODE route)
  • Game image files (.bin/.cue) or correctly burned backups

How it works — the process

This walkthrough names the tools at each stage so you know what each one is and what it's for. Always use the latest release of each named tool.

  1. Identify hardware
    Open the console and determine the motherboard revision; this dictates whether the swap trick works, which modchip wiring applies, and which ODE fits.
  2. Choose a method
    A solderable modchip (PSNee/MM3) for cheap CD-R booting, or an ODE (xStation/MODE) for disc-free SD loading and best longevity.
  3. Install the chip or ODE
    Solder the modchip to the documented points (plus PAL BIOS pins), or install the ODE per its revision-specific guide. Keep wires short.
  4. Prepare media
    Burn backups to CD-R (patch if needed), or load .bin/.cue images onto the ODE's SD card and update its firmware.
  5. Test and verify
    Boot a known-good backup or image; confirm out-of-region titles run and CD audio plays correctly.

Which option should I pick?

Want disc-free convenience and to spare the laser? Get an ODE (xStation is the value pick, MODE the premium all-in-one). Just want cheap backup booting and can solder? A PSNee is the go-to. Only try the swap trick on the earliest fat consoles.
