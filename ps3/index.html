PlayStation 3 Hacks Guide

Run homebrew, emulators, game backups, and system mods via Custom Firmware (CFW) or the Homebrew Enabler (HEN).

What this is for

Modding a PS3 lets you run unsigned software the console normally blocks - homebrew apps, emulators, backup managers, save editors, themes, and tweaks. There are two routes, and which one you can use depends entirely on your console model and firmware, so identify your hardware first.

⚙️ Methods, versions & compatibility — read this first

MethodWorks onNotes
Custom Firmware (CFW)
CFW		All Fat models, and Slim up to CECH-25xxPermanent and most capable. Maintained builds like Evilnat track the latest firmware. Not available on later Slims or any Super Slim.
PS3HEN (Homebrew Enabler)
Exploit		All models - required for Slim CECH-30xx and all Super Slim CECH-40xxRe-enable after each reboot. Runs on the latest firmware via HFW (Hybrid Firmware).

What to check first: The latest official firmware is the 4.9x series. CFW-capable models can install permanent CFW; everything else uses PS3HEN on HFW. Find your exact model on the sticker underneath (begins with CECH-) and check your firmware under Settings - System - System Information before choosing a path.

⚠️ Before you start: Console modding carries real risk — an interrupted install or the wrong method/file for your exact model can leave hardware unbootable. Identify your model first, back up anything important, and proceed at your own risk.

What you'll need

  • Your PS3, controller, and a stable power source
  • A FAT32-formatted USB flash drive
  • Your exact model number (CECH-____) and firmware version
  • An internet connection on the PS3 for the firmware/HEN files

How it works — the process

This walkthrough names the tools at each stage so you know what each one is and what it's for. Always use the latest release of each named tool.

  1. Identify your model & firmware
    Read the CECH- number under the console and your firmware in System Information; this decides CFW vs HEN.
  2. Back up your data
    Copy saves and important content off the console first so a failed step can't cost you anything.
  3. Choose your path
    CFW if your model supports it (permanent, most features) - otherwise PS3HEN, which every PS3 can run.
  4. Prepare the firmware
    HEN path: install HFW (Hybrid Firmware) matching the latest version. CFW path: install a maintained CFW such as Evilnat for your model.
  5. Enable homebrew
    HEN path: install and run PS3HEN (re-enable after each reboot). CFW path: it's already active and permanent.
  6. Set up homebrew
    Install a manager like webMAN MOD, multiMAN, or IRISMAN, plus emulators and tools.

Which option should I pick?

Older Fat or early Slim (CECH-25xx or lower): choose CFW - permanent and most capable. Later Slim (CECH-30xx) or any Super Slim (CECH-40xx): HEN is your only option and covers most needs. Identify your model first.
