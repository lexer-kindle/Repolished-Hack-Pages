# Contributing to Hack0Guide

Thank you for helping improve the archive! To maintain accuracy and community safety, please follow these guidelines:

## Guidelines
1. **No Piracy:** Do not submit links to copyrighted game ROMs, console keys, or proprietary system firmware. Only link to open-source homebrew tools, payloads, and official documentation.
2. **Test Your Steps:** Ensure any guide modifications are thoroughly tested. Misconfigured steps can permanently brick consoles.
3. **Credit the Original Authors:** If porting an updated step from `hacks.guide` or `cfw.guide`, ensure the proper credits are preserved.

## Submitting Changes
- Create a clear, descriptive branch name.
- Open a Pull Request and detail exactly what changed.
